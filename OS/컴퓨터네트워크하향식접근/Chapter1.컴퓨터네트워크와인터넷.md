# 컴퓨터 네트워크하향식 접근 1장

- 종단시스템
  - 통신 링크 / 패킷스위치 네트워크로 연결
  - 링크
    - 동축케이블 / 구리선 / 광케이블 / 라디오 스펙트럼 등등을 포함한 물리 매체로 구성
    - 각 링크는 다양한 전송률(링크 대역폭)을 이용
    - 전송률은 초당 비트 수를 의미하는 bps 단위를 사용
  - 송신종단시스템
    - 데이터를 ‘세그먼트’로 나누고 각 세그먼트에 ‘헤더’를 붙인다. 
    - 해당 정보 패키지를 ‘패킷’
- 패킷 교환기 / 스위치
  - 라우터와 링크 계층 스위치가 존재
  - 최종 목적지 방향으로 패킷을 전달한다.
  - 패킷이 송신 -> 수신 종단시스템까지 이동한 패킷 스위치를 네트워크상의 ‘경로’라고 함
- 소켓 인터페이스
  - 인터넷에 접속된 종단 시스템들이 인터넷 인프라스트럭처에 다른 종단 시스템에의 특정 목적지 프로그램으로 데이터를 전달하도록 요구하는지를 명시
